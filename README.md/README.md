\# Work Process Tracker



\## Description

This is a web application to track tasks. Users can add, update, and delete tasks easily.



\## Features

\- Add tasks

\- Edit tasks

\- Delete tasks

\- Mark tasks as completed



\## Technologies Used

\- React

\- Node.js

\- Express

\- MongoDB



\## Setup Instructions



\### Backend

1\. cd backend

2\. npm install

3\. npm start



\### Frontend

1\. cd frontend

2\. npm install

3\. npm start



\## Notes / Decisions

\- Using MongoDB for simplicity

\- Chose React for frontend due to fast rendering
